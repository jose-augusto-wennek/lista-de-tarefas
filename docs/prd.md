# Descrição

A aplicação tem como objetivo permitir que usuários gerenciem suas tarefas diárias de forma simples e eficiente. O sistema resolve o problema de organização pessoal, permitindo o registro, acompanhamento e conclusão de atividades.

---

#  Atores do Sistema

- Visitante (usuário não autenticado)

---

# Histórias de Usuário

- Como usuário, quero adicionar uma nova tarefa para organizar minhas atividades.
- Como usuário, quero visualizar minhas tarefas em uma lista para acompanhar o que preciso fazer.
- Como usuário, quero marcar uma tarefa como concluída para saber o que já finalizei.
- Como usuário, quero remover uma tarefa para excluir atividades desnecessárias.
- Como usuário, quero filtrar tarefas (todas, concluídas, pendentes) para melhorar minha organização.
- Como usuário, quero que minhas tarefas sejam salvas no navegador para não perder os dados ao atualizar a página.
- Como usuário, quero acessar o sistema pelo celular para gerenciar tarefas em qualquer lugar.
