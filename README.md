# Lista de Tarefas

## Descrição
Aplicação web para gerenciamento de tarefas com armazenamento local.

## Objetivo
Auxiliar usuários na organização de atividades diárias.

## Funcionalidades
- Adicionar tarefas
- Remover tarefas
- Marcar como concluída
- Filtrar tarefas

## Tecnologias
- HTML
- CSS
- JavaScript
- Framework CSS: Bootstrap 5.3

O projeto utiliza o Bootstrap como framework CSS principal devido aos seguintes fatores:

- Alta responsividade com sistema de grid flexível
- Documentação extensa e de fácil compreensão
- Grande comunidade e suporte
- Componentes prontos (navbar, cards, modais), acelerando o desenvolvimento
- Compatibilidade com diversos navegadores

Do ponto de vista visual e comercial, o Bootstrap permite criar interfaces modernas rapidamente, reduzindo o tempo de desenvolvimento e garantindo consistência no layout.

## API Pública: OpenWeather API

Foi escolhida a API OpenWeather para integração de dados externos.

Motivos da escolha:

- Fornece dados climáticos em tempo real
- Fácil integração com requisições HTTP
- Plano gratuito disponível
- Documentação clara e exemplos práticos

Essa API agrega valor ao sistema ao fornecer dados dinâmicos e atualizados, tornando a aplicação mais útil e interativa para o usuário final.
